```
Cute kawaii children's book illustration, flat color, bright saturated colors, simple shapes, white background, no text, no words, digital art, preschool style, friendly and warm
```

**Aspect ratios:**
- Character portraits: `1:1`
- Scene/activity images: `4:3`
- Banner/header images: `16:9`
- Tall cards: `3:4`

## 3. Prompt Engineering Tips

1. **Always start with style**: "Cute kawaii children's book illustration" anchors the style
2. **"flat bright colors"** prevents realistic/3D renders
3. **"white background"** for icon-style images; describe scene BG for scenes
4. **"no text, no words, no letters"** — add if text keeps appearing
5. **"simple composition"** prevents cluttered outputs
6. **Character consistency**: Use the exact same description each time (copy from §1). For better consistency, generate a character sheet first, then use image-to-image (i2i) with the reference
7. **Avoid**: "scary", "dark", "shadow", "realistic", "3D render", "photo"

## 4. Python Generation Script

Save as `preschool/scripts/generate_images.py`:

```python
#!/usr/bin/env python3
"""MiniMax Image Generation for K1 Preschool Website."""

import os, json, base64, time, requests
from pathlib import Path

API_KEY = os.environ.get("MINIMAX_API_KEY", "")
ENDPOINT = "https://api.minimax.io/v1/image_generation"
OUTPUT_DIR = Path(__file__).parent.parent / "generated_images"
OUTPUT_DIR.mkdir(exist_ok=True)

ASPECT_MAP = {"1:1": "1:1", "4:3": "4:3", "16:9": "16:9", "3:4": "3:4"}

STYLE_PREFIX = (
    "Cute kawaii children's book illustration, flat color, bright saturated colors, "
    "simple shapes, digital art, preschool style, friendly and warm, no text no words, "
)

# All prompts from the plan
PROMPTS = {
    # Day 1
    "d1_bobo_welcome": ("1:1", "baby panda character waving hello, round body, big sparkling eyes, rosy cheeks, wearing a small red scarf, standing in a sunny meadow with flowers, welcoming pose with arms open"),
    "d1_forest_party": ("4:3", "forest party scene, green trees with bunting flags, several cute animal characters wearing name badge stickers, picnic blanket on grass, balloons, sunny day"),
    "d1_red_fruits": ("4:3", "three fruits on white background: shiny red apple, red strawberry with green leaves, red cherries on a stem, kawaii faces on each fruit, bright red"),
    "d1_red_nature": ("4:3", "red flowers and red autumn leaves, bright red roses, red tulips, red maple leaves, white background"),
    "d1_red_clothes": ("4:3", "clothing items all in bright red: red dress, red hat, red shoes, red mittens, white background"),
    # Day 2
    "d2_yellow_sun": ("1:1", "kawaii sun character, happy smiling face, bright yellow rays, rosy cheeks, sparkling eyes, light blue sky background"),
    "d2_yellow_objects": ("4:3", "four yellow objects on white background: yellow banana, yellow star, yellow rubber duck, yellow taxi car, each with kawaii face"),
    "d2_body_parts": ("4:3", "labeled body parts for kids: pair of eyes, nose, smiling mouth, ear, hand, foot, grid on white background, educational"),
    "d2_dancing": ("1:1", "baby panda character dancing happily, arms and legs spread, musical notes floating, joyful, spotlight effect"),
    # Day 3
    "d3_ocean_scene": ("16:9", "underwater ocean scene, bright blue water, coral reef, seaweed, small colorful fish, bubbles, sandy bottom, sunlight streaming"),
    "d3_blue_sea_animals": ("4:3", "three sea animals on light blue background: friendly blue fish, round blue whale spouting water, smiling blue dolphin jumping"),
    "d3_shapes": ("4:3", "three geometric shapes on white background: bright red circle, blue square, yellow triangle, each with cute kawaii smiling face, bold colors"),
    "d3_shape_characters": ("4:3", "three animal characters: round green turtle, square-ish gray elephant, fox with triangular ears and tail, white background"),
    "d3_shape_matching": ("4:3", "shape matching game, colorful locks in circle square triangle shapes with matching keys, wooden table background, puzzle"),
    # Day 4
    "d4_fruit_trees": ("4:3", "three fruit trees: apple tree with 3 red apples, orange tree with 4 oranges, banana tree with 5 bananas, green grass, blue sky"),
    "d4_counting_1to5": ("16:9", "counting 1 to 5, each number next to matching quantity of colorful stars, white background, educational"),
    "d4_circle_objects": ("4:3", "five circle objects on white background: colorful beach ball, round wall clock, bicycle wheel, white plate, gold coin, kawaii faces"),
    "d4_circle_collage": ("1:1", "collage of many circle shapes in different bright colors and sizes overlapping, polka dots and stripes, playful, white background"),
    # Day 5
    "d5_carnival": ("16:9", "outdoor carnival festival, colorful tents and booths, balloons, bunting flags, cute animal characters playing, ferris wheel, sunny"),
    "d5_trophy": ("1:1", "golden trophy cup with big star, colorful confetti and ribbons, sparkles, white background"),
    "d5_photo_album": ("4:3", "open photo album showing small polaroid photos of animals shapes fruits colors, scattered stickers, wooden desk"),
    "d5_celebration": ("4:3", "celebration scene, baby panda and animal friends cheering, confetti, balloons, streamers, party hats, joyful"),
    # UI
    "ui_sticker_board": ("4:3", "sticker collection board, grid of empty circles with a few cute star and animal stickers, reward chart, white background"),
    "ui_progress_bar": ("16:9", "progress bar as colorful train on track, 5 stations with flags, cheerful, white background"),
    "ui_star_reward": ("1:1", "golden star with sparkles and happy kawaii face, glowing, confetti, white background"),
    "ui_forest_bg": ("16:9", "forest background, friendly green trees, rolling hills, bright blue sky, fluffy clouds, flowers on grass, no characters, pastel"),
}


def generate_image(image_id: str, aspect: str, prompt: str) -> str | None:
    """Generate one image, save as PNG, return file path."""
    full_prompt = STYLE_PREFIX + prompt
    if len(full_prompt) > 1500:
        full_prompt = full_prompt[:1500]

    resp = requests.post(
        ENDPOINT,
        headers={"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"},
        json={
            "model": "image-01",
            "prompt": full_prompt,
            "aspect_ratio": aspect,
            "response_format": "b64_json",
            "n": 1,
        },
        timeout=120,
    )
    resp.raise_for_status()
    data = resp.json()

    if data.get("base_resp", {}).get("status_code") != 0:
        print(f"  ERROR {image_id}: {data.get('base_resp', {}).get('status_msg')}")
        return None

    b64 = data["data"]["image_list"][0]["b64_image"]
    out_path = OUTPUT_DIR / f"{image_id}.png"
    out_path.write_bytes(base64.b64decode(b64))
    print(f"  ✅ {out_path}")
    return str(out_path)


def main():
    if not API_KEY:
        print("Set MINIMAX_API_KEY env var first")
        return

    results = {}
    for i, (image_id, (aspect, prompt)) in enumerate(PROMPTS.items()):
        print(f"[{i+1}/{len(PROMPTS)}] Generating {image_id}...")
        path = generate_image(image_id, aspect, prompt)
        results[image_id] = path
        time.sleep(2)  # rate limit buffer

    # Save manifest
    manifest = OUTPUT_DIR / "manifest.json"
    manifest.write_text(json.dumps(results, indent=2))
    print(f"\nDone! {sum(1 for v in results.values() if v)} / {len(results)} images generated.")
    print(f"Manifest: {manifest}")


if __name__ == "__main__":
    main()
```

4. If MiniMax adds image-to-image (i2i) support, use character sheets as reference
5. Alternative: Generate all images for one character in a single batch to maximize style consistency
