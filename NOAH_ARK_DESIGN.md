# 挪亞方舟遊戲設計方案

## 故事結構

### 第一日：神的警告
- 挪亞聽從神的命令
- 建造方舟
- 互動：用手指滑動建造方舟

### 第二日：動物登船
- 各類動物一對對走入方舟
- 互動：拖曳動物到方舟內

### 第三日：大雨降臨
- 烏雲密布，雨開始落下
- 互動：點擊雲朵落雨

### 第四日：洪水氾濫
- 水位上升，萬物被淹沒
- 互動：讓方舟漂浮

### 第五日：鴿子探路
- 放出鴿子探查陸地
- 互動：點擊放出鴿子

### 第六日：彩虹之約
- 彩虹出現，神的約定
- 互動：點擊彩虹，完成！

## 技術實現

### 遊戲引擎
- 單一 HTML 檔案（creation_game.html 模式）
- JavaScript 處理所有遊戲邏輯
- CSS 動畫和過場效果

### 音頻
- 使用 MiniMax TTS 生成粵語音頻
- GitHub Raw URL 托管音頻

### 圖片
- 使用 MiniMax Image 生成
- 上传到 Google Drive 或直接嵌入
