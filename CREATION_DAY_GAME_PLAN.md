# 創造七日遊戲設計
# Genesis Creation Week Game Design
# 最終版 2026-04-02

---

## 📖 遊戲概覽

| 項目 | 內容 |
|------|------|
| 總頁數 | 6（日 + 6個創造日）|
| 目標年齡 | K1-K2（3-5歲）|
| 主題 | 創世紀 1-2章 創造天地 |
| 語言 | 粵/普/EN/JA 四語言 |
| 角色 | 旁白（無名字）+ 聖經人物 |

---

## 🎮 遊戲流程

### 第0頁：歡迎頁
| 元素 | 內容 |
|------|------|
| 標題 | 神的創造 |
| 旁白說話 | 「今日我哋一齊聽聖經故事！神用六日創造世界，我哋一齊睇吓第一日創造啲乜！」 |
| 選項 | 【⭕ 幫手】 / 【❌ 唔幫手】 |
| ⭕ 回應 | 「多謝你願意幫手！我哋一齊聽聖經故事！」 |
| ❌ 回應 | 「唔緊要，下次再搵你幫手啦！Bye Bye！」 |

### 第1日：光
| 元素 | 內容 |
|------|------|
| 聖經經文 | 創1:3「神說，要有光就有了光」 |
| 遊戲方式 | 🎤 語音互動：對著咪高峰大聲說話，聲音越大=光越亮 |
| 進度條 | 顯示音量大小，觸發光出現 |
| 成功條件 | 音量達到一定大小 |
| 成功回饋 | 「啱啦！光係第一日創造嘅！你令世界光亮喇！」 |
| 失敗回饋 | 「再大聲少少！光要更大聲喔！」 |
| 家長提示 | 「這個遊戲訓練幼兒口腔肌肉控制和語音爆發力，是語言發展的重要基礎。」 |

**音頻腳本：**
| ID | 粵語 | 普通話 | EN | JA |
|----|------|--------|-----|-----|
| cr_d1_intro | 神說：要有光，我哋一齊試下大聲創造光！ | 神說：要有光，我們一起試下大聲創造光！ | God said let there be light, let's try creating light with our voice! | 神は光あれと言われた、声を大きくして光を創造しよう！ |
| cr_d1_correct | 啱啦！光係第一日創造嘅！你令世界光亮喇！ | 對了！光是第一日創造的！你令世界光亮喇！ | Correct! Light was created on Day 1! You made the world bright! | せいかい！光は初日に創造された！世界を明るくしたね！ |
| cr_d1_wrong | 再大聲少少！光要更大聲喔！ | 再大聲少少！光要更大聲喔！ | Louder! The light needs more sound! | もっと大きな声で！光はもっと音が必要！ |

### 第2日：天空
| 元素 | 內容 |
|------|------|
| 聖經經文 | 創1:6-8「神說：諸水之間隔要有空氣」 |
| 遊戲方式 | ☁️🐟 拖曳互動：向上拖=雲（放天空），向下拖=水（放海洋） |
| 畫面 | 左邊：陸地；右邊：混沌的水；上方：天空 |
| 遊戲目標 | 將雲拖到天空上方，水拖到陸地周圍 |
| 成功回饋 | 「啱啦！天喺上面，水喺下面，天空就係分開天地的空氣！」 |
| 錯誤回饋 | 「再留意吓，雲係飘上面，水係流下面喔！」 |
| 家長提示 | 「這個遊戲訓練幼兒理解上/下的空間概念，是方位知覺發展的重要部分。」 |

**音頻腳本：**

| cr_d2_correct | 啱啦！天喺上面，水喺下面！ | 對了！天在上面，水在下面！ | Correct! Sky is up, water is down! | せいかい！空は上で、水は下！ |

|------|------|
