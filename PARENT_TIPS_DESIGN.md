# 知心一點點 — 家長提示

## 定位

唔係文章，係**簡潔提示**。
配合每個活動頁面，家長隨時可以查看/關閉。

---

## 呈現方式

### 活動頁面提示按鈕
```
[💡 家長提示]          ← 點擊展開
```

### 提示內容
```
┌─────────────────────────────────────┐
│  💡 知多一點點                       │
│                                     │
│  這個年紀的孩子正在發展...          │
│                                     │
│  📚 出處：香港教育局 K1-K3指引      │
│                                     │
│                    [關閉]            │
└─────────────────────────────────────┘
```

---

## 提示格式

| 欄位 | 內容 |
|------|------|
| 標題 | 簡潔標題 |
| 內容 | 1-2句要點 |
| 出處 | 引用來源 |
| 建議 | 實際做法 |

---

## 出處範例

| 來源 |
|------|
| 香港教育局 K1-K3 課程指引 |
| WHO 幼兒身體活動指引 |
| Piaget 認知發展論 |
| Montessori 教學法 |
| 美國兒科學會 |

---

## 代碼實現

### HTML
```html
<div id="parent-tip" class="tip-box">
  <button class="tip-toggle" onclick="toggleTip()">💡 家長提示</button>
  <div class="tip-content" style="display:none">
    <h4>💡 知多一點點</h4>
    <p id="tip-text"></p>
    <cite id="tip-source"></cite>
    <button onclick="toggleTip()">關閉</button>
  </div>
</div>
```

### 提示數據
```javascript
const tips = {
  creation_day1: {
    text: "2-3歲幼兒對光暗敏感，可藉此機會教導分辨晝夜。",
    source: "香港教育局 K1-K3 課程指引"
  },
  noah_day2: {
    text: "配對遊戲有助幼兒建立分類概念，為數學打基礎。",
    source: "Piaget 認知發展論"
  }
};
```

## 示例提示

### 創造 Day 1（光）
> 這個年紀的孩子正在建立時間概念，光與暗的體驗有助他們理解晝夜交替。
> 📚 出處：香港教育局 K1-K3 課程指引

### 挪亞 Day 2（動物）
> 2-4歲幼兒喜歡重複動作，數數活動配合動物主題能增加學習興趣。
> 📚 出處：Montessori 教學法

### 摩西 Day 3（祈禱）
> 模仿動作有助粗大動作發展，同時建立正確的身體意識。
> 📚 出處：WHO 幼兒身體活動指引
