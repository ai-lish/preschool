# 👶 幼兒學習遊戲

## 用途
為2-4歲小朋友設計既互動學習遊戲，適合iPad使用。

## 遊戲內容

### 🎨 畫畫時間
- 圓形、正方形、三角形
- 用手指跟住灰色虛線畫
- 完成後有celebration

### 👦👧 男仔女仔
- 認識「男」「女」兩個字
- 將貼紙拖去啱既位置
- 琪琪係女仔，俊俊係男仔
- 點擊貼紙再點擊空格放置

## 功能
- 📱 iPad觸控優化
- ⭐ 完成後有celebration鼓勵
- 🔄 可無限次重複
- 🏠 目錄式介面

## 使用方法
用Safari打開 `index.html`，小朋友可以：
1. 喺目錄選擇遊戲
2. 完成後點「完成」或「檢查」
3.睇celebration獎勵
4. 可以返去再玩其他遊戲
