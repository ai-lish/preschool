# 🌱 幼兒聖經故事平台

為 K1-K3（3-6歲）幼兒設計的聖經故事互動學習平台，以**廣東話**主要語言，透過遊戲學習聖經故事和品格教育。

**線上試玩：** https://ai-lish.github.io/preschool/

### 創造七日遊戲

| 日 | 主題 | 互動方式 |
|----|------|----------|
| Day 0 | 歡迎頁 | 揀語言、開始/離開 |
| Day 1 | 光 | 大聲講「要有光」出現光 |
| Day 2 | 天空 | 拖雲上天空、拖水落大海 |
| Day 3 | 陸地植物 | 點水聚水、陸地浮現、點擊發芽 |
| Day 4 | 太陽月亮星星 | 太陽→白天、星星→黑夜 |
| Day 5 | 魚和鳥 | 撈魚、搖樹搵雀 |
| Day 6 | 動物和人 | 點擊動物、對泥土吹氣創造亞當夏娃 |

### 摩西過紅海遊戲

| 日 | 主題 | 互動方式 | 學習活動 |

## 🧪 測試清單（creation_game.html）

### 基本功能測試

- [ ] **語言切換**：4語言（粵/普/英/日）切換正常，文字正確更新
- [ ] **導航欄**：🏠 返回首頁、🌐 語言選單、🔊 重播 narration
- [ ] **進度條**：完成每日後正確更新百分比

### Day 0 - 歡迎頁

- [ ] 語言正確切換（粵/普/英/日）
- [ ] 「幫手/唔幫手」按鈕正確跳轉到 Day 1
- [ ] Narration 文字正確

### Day 1 - 光

- [ ] 咪高峰權限請求正常
- [ ] 音量偵測正常（說話時音量條有反應）
- [ ] 說話大聲後 ☀️ 出現，背景變亮
- [ ] 長按 fallback（當咪高峰無法使用時）
- [ ] 跳過按鈕正常運作
- [ ] 成功後顯示 ✅ + narration + Next 按鈕
- [ ] 進度條更新

### Day 2 - 天空

- [ ] 雲朵 ☁️ 可拖曳
- [ ] 水滴 💧 可拖曳
- [ ] 拖雲到天空區域 → 顯示 ✅
- [ ] 拖水滴到地面區域 → 顯示 ✅
- [ ] 兩者完成後 → 成功反饋 + Next 按鈕
- [ ] 拖錯位置 → 顯示失敗提示 + 回到原位

### Day 3 - 陸地與植物

- [ ] 點擊水面 → 水聚攏，陸地浮現
- [ ] 點擊陸地 → 植物發芽
- [ ] 點擊果樹 → 水果落下
- [ ] 完成後 → 成功反饋 + Next 按鈕

### Day 4 - 太陽月亮星星

- [ ] ☀️ 拖到日間區域 → 放置成功
- [ ] 🌙 拖到夜間區域 → 放置成功
- [ ] ⭐⭐ 拖到夜間區域 → 放置成功
- [ ] 三者完成後 → 成功反饋 + Next 按鈕
- [ ] 拖錯位置 → 失敗提示

### Day 5 - 魚和鳥

- [ ] 點擊魚網 🎣 → 魚出現
- [ ] 點擊樹 🌳 → 鳥出現
- [ ] 兩者完成後 → 成功反饋 + Next 按鈕

### Day 6 - 動物和人

- [ ] 8個動物依次出現動畫
- [ ] 點擊每個動物 → 播放音效 + 顯示名稱
- [ ] 點擊泥土 🪨 → 亞當 🧍 出現
- [ ] 出現「吹氣」按鈕
- [ ] 吹氣 3 次 → 亞當睡覺 😴
- [ ] 夏娃 👩 出現
- [ ] 完成後 → 成功反饋 + 完成按鈕
