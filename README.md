# 🌱 K1 成長學習平台

為 K1（3-4歲）小朋友設計的互動學習平台，陪伴孩子成長。

---

## 📋 製作階段

```
Phase 0 ─── 確認框架 ────────────────── ✅ 完成
Phase 1 ─── 技術架構 ────────────────── ✅ 完成
Phase 2 ─── Week 1 活動設計 ──────────── ▶ 即將開始
Phase 3 ─── Week 1 開發 ────────────── 待做
Phase 4 ─── Week 1 測試 ────────────── 待做
Phase 5 ─── Week 2-4 擴展 ──────────── 待做
Phase 6 ─── 家長Dashboard ──────────── 待做
```

---

## 🎯 學習範圍

### 語文
- 聆聽理解
- 常用字認識
- 口語表達

### 數學
- 1-10 數數
- 顏色認識
- 形狀認識
- 大小概念

### 英文
- 字母認識
- 簡單詞彙

### 社交/情緒
- 認識自己
- 家庭成員
- 分享概念
