# 👶 幼兒學習遊戲

為2-4歲小朋友設計既互動學習遊戲，適合iPad使用。

## 👶 幼兒語文

### 🎨 畫畫
- 圓形、正方形、三角形
- 用手指跟住灰色虛線畫

### 👦👧 男仔女仔
- 認識「男」「女」兩個字
- 將貼紙拖去啱既位置

## 🔢 幼兒數學

### 1️⃣2️⃣3️⃣ 數字
- 認識1-10既數字同中文讀法

### 🎨 顏色
- 紅、藍、綠、黃、橙、紫

### 🐘🐜 大與小
- 比較大小物件

## 功能
- 📱 iPad觸控優化
- ⭐ 完成後有celebration鼓勵
- 🔄 可無限次重複
- 🏠 目錄式介面

用Safari打開 https://math-lish.github.io/preschool/
