# 🌱 幼兒聖經故事平台

為 K1-K3（3-6歲）幼兒設計的聖經故事互動學習平台，以**廣東話**主要語言，透過遊戲學習聖經故事和品格教育。

**線上試玩：** https://ai-lish.github.io/preschool/

### 創造七日遊戲

| 日 | 主題 | 互動方式 |
|----|------|----------|
| Day 0 | 歡迎頁 | 揀語言、開始/離開 |
| Day 1 | 光 | 大聲講「要有光」出現光 |
| Day 2 | 天空 | 拖雲上天空、拖水落大海 |
| Day 3 | 陸地植物 | 點水聚水、陸地浮現、點擊發芽 |
| Day 4 | 太陽月亮星星 | 太陽→白天、星星→黑夜 |
| Day 5 | 魚和鳥 | 撈魚、搖樹搵雀 |
| Day 6 | 動物和人 | 點擊動物、對泥土吹氣創造亞當夏娃 |

### 挪亞方舟遊戲 ✅

| 日 | 主題 | 互動方式 |
|----|------|----------|
| Day 1 | 建造方舟 | 點擊 8 粒釘子建造方舟 |
| Day 2 | 動物登船 | 拖卡通動物到方舟，一對對上船（8對） |
| Day 3 | 大雨降臨 | 點擊 4 朵雲落雨 |
| Day 4 | 洪水氾濫 | 按鈕讓方舟漂浮 |
| Day 5 | 鴿子探路 | 放出鴿子🕊️ |
| Day 6 | 彩虹之約 | 點擊彩虹🌈完成 |

**Day 2 卡通動物 + 叫聲 ✅**
- 8 種卡通動物：大象/獅子/狼/羊/牛/馬/兔/鹿
- 拖一對上船 → 播放該動物叫聲（4語言）
- 叫聲目錄：`audio/noah/sounds/{lang}/{animal}.mp3`

### 摩西過紅海遊戲

| 日 | 主題 | 互動方式 | 學習活動 |

## 🧪 測試清單（noah_game.html）

### 基本功能測試

- [ ] **語言切換**：4語言（粵/普/英/日）🌐 切換正常，文字正確更新
- [ ] **導航欄**：🏠 返回首頁、🌐 語言選單、🔊 重播 narration
- [ ] **進度條**：完成每日後正確更新百分比

### Day 1 - 建造方舟

- [ ] 顯示方舟 + 8 粒釘子
- [ ] 點擊釘子 → 釘子消失、進度更新
- [ ] 完成 8 粒 → 方舟建成動畫、成功提示
- [ ] 顯示「第二日 ▶」按鈕

### Day 2 - 動物登船 ⭐ 重點測試

- [ ] **顯示 16 張卡通動物圖**（8種 × 2張）
- [ ] **拖曳功能**：動物可拖動
- [ ] **方舟 Drop Zone**：方舟喺右邊，拖過去高亮綠色
- [ ] **配對上船**：拖任意一張 → 該物種的兩張同時變暗
- [ ] **叫聲播放**：上船嗰時播放該動物叫聲（4語言都試）
- [ ] **語言跟進**：粵語→粵語叫聲、普→普通話、EN→English、日→日本語
- [ ] **方舟內顯示**：成功上船的動物圖顯示喺方舟入面
- [ ] **Feedback 文字**：「大象 x2」「獅子 x2」等正確顯示
- [ ] **防重複**：已上船的 pair 再拖無反應
- [ ] 完成 8 對 → Day 2 完成

### Day 3 - 大雨降臨

- [ ] 4 朵雲朵可點擊
- [ ] 點擊 → 雨滴動畫落下、雲變暗
- [ ] 完成 4 朵 → 成功提示、顯示「第三日 ▶」

### Day 4 - 洪水氾濫
