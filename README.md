# 👶 幼兒學習遊戲

為2-4歲小朋友設計既互動學習遊戲，適合iPad及手機使用。

## 📖 幼兒語文

### 第 1 週 — 男孩子和女孩子

**學習重點：**
- 辨識字詞 — 男孩子、女孩子
- 認識筆順 — 橫 (一)
- 學習句式 — 我是男孩子。

**遊戲：**

| 遊戲 | 內容 |
|------|------|
| 👦👧 男女配對 | 將「男」「女」字拖入正確句子位置 |
| 🎀 顏色配對 | 將相同顏色蝴蝶結配對 → 跟隨虛線連線 |
| ✏️ 女字書寫 | 鷹架學習女字三筆筆順 |
